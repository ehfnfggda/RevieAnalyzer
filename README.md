# Анализ тональности отзывов

Веб-приложение для автоматического анализа отзывов покупателей маркетплейсов (Ozon, Wildberries и других). Определяет тональность каждого отзыва (позитивный / негативный / нейтральный) и группирует результаты по темам: доставка, качество, цена, сервис, упаковка.

---

## Как это работает

1. Загружаете CSV-файл с отзывами (столбец `text`)
2. Нажимаете «Запустить анализ»
3. Получаете диаграммы, статистику по темам и возможность фильтровать отзывы

Для классификации тональности используется предобученная модель **RuBERT** (`blanchefort/rubert-base-cased-sentiment`) - русскоязычная версия BERT.

---

## Установка и запуск

```bash
# 1. Клонируйте репозиторий
git clone https://github.com/ВАШ_НИК/ozon-reviews-analysis.git
cd ozon-reviews-analysis

# 2. Установите зависимости
pip install -r requirements.txt

# 3. Запустите приложение
streamlit run dashboard.py
```

Приложение откроется в браузере по адресу `http://localhost:8501`

---

## Формат входного файла

CSV-файл с обязательным столбцом `text`:

```
text
Отличный товар, доставили быстро!
Качество плохое, пришёл с браком.
Нормально, ничего особенного.
```

---

## Настройка тем

Словарь ключевых слов хранится в файле `themes.json`. Можно добавлять новые темы или расширять существующие без изменения кода:

```json
{
  "Доставка": ["доставка", "курьер", "посылка", ...],
  "Качество":  ["качество", "брак", "дефект", ...]
}
```
